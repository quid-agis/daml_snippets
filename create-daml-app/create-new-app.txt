This is a simple tutorial on creating a basic Daml application, using an existing template.

#### Select Daml Application Build Directory
$ cd ~/Projects/

#### Create the App
$ daml new create-daml-app --template create-daml-app

#### Change into the new Application
$ cd create-aml-app

#### Inspect the Application structure
$ tree
